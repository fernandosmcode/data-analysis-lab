# :abacus: Calculadora de IMC

Projeto desenvolvido em Portugol como exercício prático do curso de Algoritmos.

---

## :pushpin: Descrição

Este algoritmo calcula o IMC (Índice de Massa Corporal) a partir dos dados informados pelo usuário.

O programa solicita:

- Nome
- Altura (em metros)
- Peso (em kg)

E realiza o cálculo utilizando a fórmula:

IMC = peso / (altura²)

Ao final, exibe o resultado junto com os dados fornecidos.

---

## ⚙️ Tecnologias Utilizadas

- Portugol
- Visualg

---

## ▶ Como Executar

1. Abrir o Visualg
2. Carregar o arquivo `calculadora-imc.alg`
3. Executar o algoritmo
4. Informar os dados solicitados
